<h1 align="center">Rong Form</h1>

> Rong Form is a modern form management library for React

<br/>
<br/>

# Quick Start

1.  Install: 
```bash
yarn add rong-form -S
```
```bash
npm i rong-form -S
```

2.  Import :

```jsx
import { Form,Field } from "rong-form";
ReactDOM.render(
  <Form>
    <Field name="username">
      <input/>
    </Field>
  </Form>,
  document.getElementById("root")
);
```
